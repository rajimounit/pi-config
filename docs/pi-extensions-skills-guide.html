C'est quoi un MCP Server ?
Model Context Protocol — standard ouvert Anthropic
Pi se connecte via pi-mcp-adapter · pi-tidy-mcp-adapter

Un MCP Server expose des outils que Pi peut appeler comme des fonctions : lire des tickets Jira, pousser sur GitHub, lancer des tests Playwright, interroger une base de données. C'est l'équivalent des plugins pour agents IA.

Priorité haute — installation recommandée
GitHub MCP
@modelcontextprotocol/server-github
★ #1 Recommandé

Le MCP le plus utilisé pour les devs. Pi peut créer/lire des issues, ouvrir des PRs, lire le CI, faire des reviews de code directement depuis la session. Intégration complète avec rajimounit/pi-config et le repo AVV Assist Pro.

# Installation via pi-tidy-mcp-adapter : pi install npm:pi-tidy-mcp-adapter # Config dans ~/.pi/agent/mcp-servers.json : { "github": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-github"], "env": { "GITHUB_PERSONAL_ACCESS_TOKEN": "..." } } }
Requiert un Personal Access Token GitHub. Stocker dans ~/.pi/.env (gitignored).
Playwright MCP
@playwright/mcp
★ Recommandé

Pi peut lancer des tests e2e Playwright, naviguer dans le navigateur, prendre des screenshots, interagir avec les formulaires. Direct pour tester la SPA AVV Assist Pro.

npx playwright install chromium # Config MCP : { "playwright": { "command": "npx", "args": ["@playwright/mcp"] } }

Cas d'usage AVV Assist Pro : Pi ouvre le front, navigue, vérifie que le wizard de qualification fonctionne, screenshot si erreur.

PostgreSQL MCP
@modelcontextprotocol/server-postgres
★ Recommandé

Pi peut interroger directement la base PostgreSQL d'AVV Assist Pro (avv-postgres-v3). Lire le schéma, exécuter des queries, vérifier les données sans passer par l'API NestJS.

{ "postgres": { "command": "npx", "args": ["-y", "@modelcontextprotocol/server-postgres", "postgresql://localhost/avv_assist"] } }
Idéal pour debug : Pi peut vérifier directement si les données sont bien en base après une opération.

Concept : AVV Assist Pro devient un MCP Server → Pi Agent peut l'orchestrer directement.

# Dans le backend NestJS : npm install @nestjs-mcp/server # Décorer les services avec @Tool()
C'est le Palier D le plus ambitieux — transformation d'AVV Assist Pro en outil natif Pi.
